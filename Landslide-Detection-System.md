# Proactive Landslide Detection Device
In this project, I'll be making a simple landslide detection device for a tech showcase.

## About this project
I built this project for a showcase at our library with a teammate. It uses an ESP32 board, an IMU sensor, a MQTT broker, and a local web for processing and displaying the data. I used an AI Coding agent to help me debug and write the code. 

### What it does
This project can detect landslides on a model slop before it becomes devestating. It sends a wireless alert on a dashboard if the model slop moved more than the threshold.

### Why I chose this project
In the area I live in, landslides are a major problem and the current tools used to detect them are ver inffefficient and expenscive.
