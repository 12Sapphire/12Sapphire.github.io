# Proactive Landslide Detection Device
In this project, I'll be making a simple landslide detection device for a tech showcase.

## _____________________________________________________________________________________________


This project can detect landslides on a model slop before it become devestating. It sends a wireless alert on a dashboard if the model slop moved more than the threshold.

This was a team project and my job was the technical stuff.

This project uses an ESP32 board, an IMU sensor, a MQTT broker, and a local website for processing and displaying the data.
