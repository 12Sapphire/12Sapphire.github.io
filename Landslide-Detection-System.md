# Proactive Landslide Detection Device

 In this project, I'll be making a simple landslide detection device for a tech showcase.

## About this project

I built this project for a showcase at our library with a teammate. It uses an ESP32 board, an IMU sensor, an MQTT broker, and a local web dashboard for processing and displaying the data. I used an AI coding agent to help me debug and write the code.

### What it does
This project can detect landslides on a model slope before they become devastating. It sends a wireless alert on a dashboard if the model slope moves more than the threshold.

### Why I chose this project
In the area I live in, landslides are a major problem, and the current tools used to detect them are very inefficient and expensive. So we developed a basic device that is cheaper and more effiecient and can detect landslides on a model slope.

### How I built it
I started by using the AI coding agent to write the code. As I faced challenges and errors, I iterated the code and adapted it to the new circunstances. The main challenges and errors were getting it to work at the showcase, sending the data wirelessy, and getting the readings correct.
